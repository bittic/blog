人类组织形式的发展

人类是一群生来自由平等的哺乳动物，但为了生存、竞争、协作，有史以来，人类发展了数种基本的组织形式，有

  • 基于血缘的家庭
  • 基于信仰的宗教
  • 基于土地的国家
  • 基于资本的公司

组织形式是互相掺杂的

这四种基本组织形式，你中有我，我中有你。例如，一个家庭，可以有不同国籍的家庭成员。一个宗教例如伊斯兰，可以拥有多个国家。一个国家，依法管辖其境内的公司法人。一个公司，可以跨国运营，甚至拥有国家作为工具。

组织形式是不断发展变化的

我们回顾一下过去500年的历史，可以看到，组织形式不是一成不变的，而是在历史进程中，伴随思想、科技的发展而不断发展、变化。

1500～1700 复式记账法，手工商业资本主义，宗教改革，文艺复兴，自然科学

16世纪初，从意大利开始了资本主义萌芽，发明了复式记账法，进入手工商业资本主义时期，同时伴随着文艺复兴运动，出现了但丁、莎士比亚、达芬奇，以及自然科学大发展，出现了哥白尼、伽利略、牛顿。

  • 1492 哥伦布，发现新大陆
  • 1494 意大利著名数学家卢卡·帕切奥利：复式记账法，从技术上为资本主义做好储备
  • 1517 马丁路德：《95条论纲》、《论基督徒的自由》等，引发了宗教改革
  • 1519 麦哲伦，环球航行
  • 1543 哥白尼：《日心说》
  • 1576 让·博丹 《国家六论》，提出国家主权概念
  • ～1600 伽利略：近代科学
  • 1687 牛顿：《自然哲学的数学原理》

1733～1800 工业革命，自由资本主义，圈地运动，启蒙运动

从英国开始了第一次工业革命，发明了纺纱机、蒸汽机。农民被大量圈地，成为雇佣工人，进入自由资本主义时期。同时产生了追求自由、平等的启蒙运动。

  • 孟德斯鸠:《论法的精神，1948》
  • 卢梭:《论人类不平等的起源和基础，1755》，《社会契约论，1762》
  • 亚当斯密:《国富论，1776》奠定了资本主义理论基础
  • 美国独立战争，发表《独立宣言，1776》
  • 法国大革命，发表《人权宣言，1789》

1850～1900 电气革命，垄断资本主义，现代公司制度，共产主义思潮

发生了电气革命，发明了电灯、发电机、内燃机、电话、汽车、飞机，进入垄断资本主义时期，出现了现代公司制度。同时诞生了共产主义思潮。

  • 在德国，马克思和恩格斯发表《共产党宣言 1848》，《资本论 1867》，拉开了共产主义思潮和运动的大幕。
  • 在英国，罗伯特罗威《合股公司法 1856》，《公司法 1862》，宣告了现代公司制度的诞生，出现了股东、职员的角色。

1900～1950 核能革命，国家资本主义（帝国主义），二次世界大战，民族解放运动，共产主义运动

进入了国家资本主义时期（帝国主义），发生了二次世界大战，促成了大量民族国家、共产主义国家的诞生。

  • 爱因斯坦:《相对论, 1905》
  • 凯恩斯:《就业、利息和货币通论, 1937》
  • 科斯:《企业的本质：交易成本论, 1937》分析了企业存在的意义和条件，预言了企业制度的消亡。

1950～2000 信息革命，资本主义vs.共产主义阵营冷战

从国家上升到资本主义和共产主义的两大阵营，进入冷战时期。同时发生了信息技术革命。

  • 科斯: 《社会成本问题：产权理论, 1960》
  • 信息革命，发明了计算机、晶体管、芯片、互联网。
    • 第一台计算机 1946，互联网前身 ARPANET 1969，万维网 1989。

可以看到，主权国家在约500年前诞生，而公司仅仅诞生了约150年，是最年轻、最富于生产力的组织形式，创造了人类财富的绝大部分。自然的，今天我们习以为常的组织形式，也不是僵化的，其发展变化不会停止。

家庭这种最常见的组织形式，也经历了多样的变化。从母系氏族，到妻妾制度，到一夫一妻小家庭制。从家庭的最基本单位出发，也可延伸扩展到家族，氏族，民族，种族。

未来的组织形式

当我们回首1000年前，在封建甚至奴隶社会，我们很同情那时的人们。当我们展望1000年后，可以绝对肯定的说，也绝对不会是我们今天的科技、生活、主义、组织方式。我们的后人回望今天，也会非常同情我们生活在水深火热中。否则，我们这一代人有何面目面对后人呢。而现代的变化速度，远大于1000年前，所以很可能，仅仅100年后，就有翻天覆地的新变化。那么，会是哪些变化呢？

  • 区块链技术指南：记账科技的千年演化
  • 中国信息通信研究院: 区块链白皮书 2018: 区块链打破了原有的复式记账， 变成“全网共享”的分布式账本)

区块链能够对每个人，进行自负其责、公开透明、不可篡改的记账，它必将催生全新的组织形式。

资本主义世界的原子是公司，区块链从消解公司开始。
